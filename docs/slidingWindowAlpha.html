How to sliding window

  • anchor at a CpG

  • window extends +/- from it

  • new alpha for each read is calculated based on the number of CpGs within the window

  • make the window length the read length

  • this sample the reads are 36bp (I know it’s crazy)

  • make a granges of windows? - and then overlap and recalculate?

windows <- for_beta %>% distinct(seqnames, start) %>%
  mutate(index = start, start = index - 18, end = index + 18) %>% as_granges() 
for_beta_window <- find_overlaps(windows, as_granges(mutate(for_beta, cpg_pos = start))) %>% data.frame() %>%
  group_by(index, read_id) %>%
  mutate(window_total = n(),
         window_pattern = paste0(meth_status, collapse = ""),
         window_meth = str_count(window_pattern, pattern = "M"),
         window_unmeth = str_count(window_pattern, pattern = "U"),
         window_alpha = window_meth/window_total) %>%
  ungroup()
